## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No additional installations required!

### Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/shivamk-tech/TaskManager.git
```

2. Navigate to the project directory:
```bash
cd TaskManager
```

3. Open `index.html` in your web browser:
   - Simply double-click the `index.html` file, or
   - Right-click and select "Open with" your preferred browser

That's it! The application will run directly in your browser.

## Usage

1. **Add a Task**: Type your task in the input field and press Enter or click the add button
2. **Complete a Task**: Click on the task or checkbox to mark it as complete
3. **Delete a Task**: Click the delete/remove button next to any task
4. **Edit a Task**: Click the edit button to modify existing tasks

Your tasks are automatically saved and will be there when you return!

## Key Highlights

### Local Storage Integration
All your tasks are automatically saved to your browser's localStorage. This means your data persists even after closing the browser or refreshing the page - no backend or database needed!

### Responsive Design
The application adapts seamlessly to any screen size, providing an optimal experience on desktop computers, tablets, and mobile devices.

### Zero Dependencies
Built with pure vanilla JavaScript - no frameworks, no libraries, no build tools. Just open and use!
